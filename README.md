# 🎓 Student Management System

## 📌 Project Overview
The Student Management System is a web-based application designed to manage student records efficiently. It allows users to perform CRUD (Create, Read, Update, Delete) operations with secure login and logout functionality.

---

## 🚀 Features
- 🔐 User Authentication (Login/Logout)
- ➕ Add new student records
- 📋 View all students
- ✏️ Update student details
- ❌ Delete student records
- 🔍 Search students by ID or Name

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS
- Backend: Java (Servlets, JDBC)
- Database: MySQL

---

## 📂 Project Structure
- `/src` → Java source files (Servlets, DAO)
- `/webapp` → HTML, CSS, JSP files
- `/database` → SQL files

---

## ⚙️ How to Run the Project
1. Clone the repository  
   git clone https://github.com/your-username/student-management-system

2. Import project into Eclipse / IntelliJ

3. Configure Apache Tomcat Server

4. Setup MySQL Database  
   - Create database  
   - Import SQL file  

5. Run the project on server
